Installation

Prior to installing this module, if you have any existing partner you should ensure they already have a unique reference (or no reference) set.
Partner with a empty reference will automatically be assigned a code of "P" followed by the system id for that partner.
Otherwise the setting of the unique constraint will fail and the module will fail to install.
