POS Quick Access

This Module use to access POS Quickly with Shortcut Menus.

We-R Informative

Features

  • User can have two different views for product list. One is list view and another one is grid view.
  • New customer can be directly created from product dcreen using shortcut buttons.
  • Product can be created from POS directly without navigate to backend inCore.
  • By one click user can clear whole cart.
Configuration for POS Quick Access

Based on configuration user can have two different views for product list.User can select grid view as well list view


User can create customer from product screen by click on shortcut menu of create customer.No need to go customer screen for create customer.


Based on POS configuration users will allowed to create product from POS from shrotcut menu.


User can clear cart by one click.
