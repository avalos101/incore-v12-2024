incore 12 POS Pricelist

Note: This module is tested and working in Enterprise Edition. And we are supporting Ubuntu OS and not Windows OS.

Features

  • Display Original Price and Discounted Price on Item while Applying Pricelist.
  • Add Shortcut Key to Open Pricelist Selection.

How to use

  • Configure New Pricelist.


  • Enable Pricelist from Point Of Sale Configuration.


  • Open Pricelist Selection by Keyboard Shortcut that we configured in Point Of Sale Configuration.
  • Use UP DOWN Keys To Move Between Pricelists.
